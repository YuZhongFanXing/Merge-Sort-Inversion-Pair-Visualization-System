# 归并排序逆序对可视化系统

## 快速开始

直接用浏览器打开 `index.html` 即可运行，无需服务器或安装任何依赖。

## 功能说明

| 功能 | 说明 |
|------|------|
| 数组输入 | 在输入框填写整数，逗号或空格分隔，建议 2-16 个元素 |
| 更新数据 | 加载输入框中的新数组 |
| 随机生成 | 随机生成 5-11 个元素的演示数组 |
| 开始执行 | 启动归并排序逐步动画 |
| 暂停/继续 | 控制动画播放状态 |
| 重置 | 清空可视化区域，返回初始状态 |
| 演示速度 | 滑块调节每步间隔（100ms - 2000ms） |
| 下载压缩包 | 下载包含完整应用的 .zip 文件 |

## 可视化说明

```
🟡 黄色方块  ── 分割阶段的子数组
🟢 绿色方块  ── 已分割到底的单元素（叶节点）
🔴 红色方块  ── 归并阶段产生的有序数组
🔵 蓝色数字  ── 每次归并发现的逆序对增量（如 +2）
-- 虚线      ── 分割阶段与归并阶段的分界线
```

## 算法原理

利用归并排序统计逆序对：

1. **分治**：递归将数组分为左右两半，直到每半只有一个元素
2. **合并**：合并两个有序子数组时，若右侧元素 `R[j]` 小于左侧当前元素 `L[i]`，
   则左侧剩余 `L.length - i` 个元素均与 `R[j]` 构成逆序对
3. **取模**：逆序对数量对 `1,000,000,007` 取模

**时间复杂度**：O(n log n)  
**空间复杂度**：O(n)
